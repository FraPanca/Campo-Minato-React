# Campo-Minato-React

## Italiano  
Sviluppo di un gioco Campo Minato contro un bot, realizzato con React.

### Descrizione  
Il gioco è suddiviso in turni e presenta due campi distinti: uno per il giocatore e uno per il bot.  

Il giocatore inizia la partita e, ad ogni turno, può selezionare una casella non ancora scelta.  

L'obiettivo è trovare consecutivamente 5 caselle sicure. Se viene selezionata una mina, la partita termina immediatamente con una sconfitta.

---

## English  
Development of a Minesweeper game against a bot, built with React.

### Description  
The game is turn-based and features two separate boards: one for the player and one for the bot.  

The player starts the match and, at each turn, can select a cell that has not been previously chosen.  

The objective is to consecutively find 5 safe cells. If a mine is selected, the game immediately ends in a loss.
